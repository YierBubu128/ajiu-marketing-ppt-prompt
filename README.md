PPT 提案指令框架 · ppt-prompt


一套专为广告 / 营销从业者设计的 AI 提案指令体系。填写项目信息，自动生成结构完整的 PPT 提案 prompt，发给任意 AI 工具即可产出高质量初稿。




这是什么

做提案最耗时的不是写内容，而是每次都要重新想框架、整理信息、跟 AI 反复沟通。

这套框架把 12 年乙方提案经验固化成一套可复用的指令模板，覆盖从品牌信息、媒介场景、竞品分析到客户隐性背景的完整输入体系。填一次，出一份能打的初稿。


适合谁用


广告公司 AE / 策略 / 媒介，需要快速产出提案初稿
品牌市场部，需要做汇报 / 述职 / 竞品分析
独立营销顾问，需要标准化自己的提案流程
包含什么

文件内容SKILL.md完整指令框架，含所有模块说明、风格库、行业备注、红线约束ppt-prompt-generator.html交互式填写工具，浏览器打开即用，自动拼合完整 prompt使用说明.md上手指南，三步开始使用


快速上手

方法一：用交互工具（推荐）


下载 ppt-prompt-generator.html
用浏览器打开（Chrome / Safari / Edge 均可，无需联网）
按 A→G 填写七个模块
点「生成完整指令」→「复制全部」
粘贴到 Claude / ChatGPT / DeepSeek，等待输出


方法二：直接用 SKILL.md

打开 SKILL.md，复制完整 Prompt 模板，按提示填写后发给 AI。


七个填写模块

A · 品牌与项目信息   ← 品牌名、受众、周期、目标
B · 品牌视觉规范     ← 风格选择 / 品牌色 HEX
C · 营销场景与平台   ← 细分场景、平台组合、内容类型
D · 行业与竞品       ← 行业、阶段、竞品清单、分析深度
E · 目标与预算       ← 核心问题、KPI、预算范围
F · 内容结构大纲     ← PPT 页面顺序与主题
G · 项目隐性背景     ← 决策结构、内部分歧、客户顾虑（选填）

G 模块是这套框架的核心差异——台面下的信息往往决定提案能不能过。填得越详细，AI 产出越贴近实际。


内置红线约束

所有生成的指令中包含以下强制约束，不可绕过：


📌 数据真实性：无来源数据必须标注「[数据待补充]」，禁止编造
📌 品牌安全：竞品描述客观中立，不得出现法律风险表述
📌 平台合规：不得出现广告法违禁词
📌 可执行性：所有建议必须在给定预算和时间内可落地
