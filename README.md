# PPT 提案指令框架 · ppt-prompt

> 一个帮广告 / 营销人做 PPT 提案的 AI 工具。把客户 brief 填进去，工具自动整合成一段完整的 AI 指令。把这段指令发给 Claude、ChatGPT、DeepSeek、Kimi 或豆包，它会帮你生成一份定制化的提案大纲，通常 10 分钟内出初稿。
>
> **注意：** 目前生成的是提案大纲，不是可以直接发给客户的成品 PPT。直接生成成品 PPT 的功能有待进一步开发，敬请期待。

---

## 适合谁用

- 广告公司 AE / 策略 / 媒介，需要快速产出比稿提案初稿
- 品牌市场部，需要做汇报 / 述职 / 竞品分析
- 独立营销顾问，需要标准化自己的提案流程

## 快速上手

**方法一：手动填写版（无需账号，直接使用）**

👉 **[点击直接打开工具](https://yierbubu128.github.io/ajiu-marketing-ppt-prompt/ppt-prompt-generator-manual.html)**

按 A→G 填写七个模块，生成完整指令，复制后发给任意 AI 工具。

智能导入功能需要调用 API，目前个人暂时无法承担公开用户的 API 费用，所以这个版本暂不支持，请见谅 🙏

**方法二：AI 智能导入版（需要 Claude 账号）**

1. 下载 `ppt-prompt-generator-ai.jsx`
2. 把文件发给 Claude，Claude 直接运行
3. 步骤 0 粘贴 brief 文字，点「AI 自动解析」自动预填 A-G 模块
4. 逐步检查补充后生成完整指令

消耗的是你自己账号的 token，不会产生额外费用。

**方法三：直接用 SKILL.md**

打开 `ppt-prompt-SKILL.md`，复制完整 Prompt 模板，按提示填写后发给 AI。

## 七个填写模块

```
A · 品牌与项目信息   ← 品牌名、受众、周期、目标
B · 品牌视觉规范     ← 风格选择 / 品牌色 HEX
C · 营销场景与平台   ← 细分场景、平台组合、内容类型
D · 行业与竞品       ← 行业、阶段、竞品清单、分析深度
E · 目标与预算       ← 核心问题、KPI、预算范围
F · 内容结构大纲     ← PPT 页面顺序与主题
G · 项目隐性背景     ← 决策结构、内部分歧、客户顾虑（选填）
```

G 模块是这套框架的核心差异——**台面下的信息往往决定提案能不能过**。填得越详细，AI 产出越贴近实际。

## 内置红线约束

所有生成的指令中包含以下强制约束，不可绕过：

- 📌 **数据真实性**：无来源数据必须标注「[数据待补充]」，禁止编造
- 📌 **品牌安全**：竞品描述客观中立，不得出现法律风险表述
- 📌 **平台合规**：不得出现广告法违禁词
- 📌 **可执行性**：所有建议必须在给定预算和时间内可落地
