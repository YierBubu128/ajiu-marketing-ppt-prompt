# PPT 提案指令框架 · ppt-prompt

> 一套专为广告 / 营销从业者设计的 AI 提案指令体系。上传客户 brief 或粘贴文字，AI 自动解析预填信息，生成结构完整的 PPT 提案 prompt，发给任意 AI 工具即可产出高质量初稿。

---

## 这是什么

做提案最累的不是写内容——是比稿。

每次比稿都是全新的品牌、全新的情况、全新的要求。你要从零搭骨架，回答 brief 里的每一个问题，平衡客户内部不同部门的立场，还要有亮点能从竞争对手里脱颖而出。

这套框架把 13 年乙方比稿经验固化成一套可复用的指令模板，覆盖从品牌信息、媒介场景、竞品分析到客户隐性背景的完整输入体系。填一次，出一份能打的初稿。

---

## 适合谁用

- 广告公司 AE / 策略 / 媒介，需要快速产出比稿提案初稿
- 品牌市场部，需要做汇报 / 述职 / 竞品分析
- 独立营销顾问，需要标准化自己的提案流程

## 快速上手

**方法一：智能导入（推荐）**

1. 下载 `ppt-prompt-generator-v2.html`，用浏览器打开
2. 步骤 0「智能导入」：上传客户 brief PDF，或直接粘贴任何文字描述
3. 点「AI 自动解析」，系统自动提取关键信息并预填到 A-G 七个模块
4. 逐步检查、补充剩余字段（尤其是 G 模块隐性背景）
5. 点「生成完整指令」→「复制全部」
6. 粘贴到 Claude / ChatGPT / DeepSeek，等待输出

**方法二：手动填写**

跳过步骤 0，直接从 A 模块开始手动填写，适合信息已在脑子里的情况。

**方法三：直接用 SKILL.md**

打开 `ppt-prompt-SKILL.md`，复制完整 Prompt 模板，按提示填写后发给 AI。

## 七个填写模块

```
A · 品牌与项目信息   ← 品牌名、受众、周期、目标
B · 品牌视觉规范     ← 风格选择 / 品牌色 HEX
C · 营销场景与平台   ← 细分场景、平台组合、内容类型
D · 行业与竞品       ← 行业、阶段、竞品清单、分析深度
E · 目标与预算       ← 核心问题、KPI、预算范围
F · 内容结构大纲     ← PPT 页面顺序与主题
G · 项目隐性背景     ← 决策结构、内部分歧、客户顾虑（选填）
```

G 模块是这套框架的核心差异——**台面下的信息往往决定提案能不能过**。填得越详细，AI 产出越贴近实际。

---

## 智能导入支持的输入方式

- **粘贴文字**：客户微信发来的需求、邮件里的项目说明、自己记录的沟通笔记，任何文字都可以
- **上传文件**：支持 PDF、TXT 格式，拖拽或点击上传

AI 解析后自动预填对应字段，缺少的内容再手动补充，大幅减少填写时间。

---

## 内置红线约束

所有生成的指令中包含以下强制约束，不可绕过：

- 📌 **数据真实性**：无来源数据必须标注「[数据待补充]」，禁止编造
- 📌 **品牌安全**：竞品描述客观中立，不得出现法律风险表述
- 📌 **平台合规**：不得出现广告法违禁词
- 📌 **可执行性**：所有建议必须在给定预算和时间内可落地
