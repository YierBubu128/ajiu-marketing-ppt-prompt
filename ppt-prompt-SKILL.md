# PPT Prompt Builder · v2

这个 skill 帮助你为每次 PPT 任务生成一套高质量的 AI 指令。

**v2 新增：智能导入功能**
使用 `ppt-prompt-generator-v2.html` 工具时，可在步骤 0 上传客户 brief PDF 或粘贴任何文字描述，AI 自动解析并预填 A-G 模块，大幅减少手动填写时间。G 模块（隐性背景）需手动补充。

---

## 使用流程

1. 打开 `ppt-prompt-generator-v2.html`（浏览器直接打开，无需联网）
2. **步骤 0（可选）**：上传 brief 文件或粘贴文字，点「AI 自动解析」预填信息
3. 按模块逐步检查、补充（A → G）
4. 点「生成完整指令」→「复制全部」
5. 粘贴到 Claude 或其他 AI 工具，等待输出
6. 做完后，将本次有效经验追加到文末「CHANGELOG」

```
# 输出目标
制作一份面向【填写：受众】的【填写：PPT类型】。

  - [ ] 其他：【填写】

- 平台组合（标注主次）：

  - 其他平台：【填写】

## H · 视觉与输出规范

视觉风格：【填写风格代号】

固定规范（无需修改）：
- 结构逻辑：SCQ框架，每页一个核心观点，先结论后支撑
- 图表优先：结构图 > 文字列表，能图不表，每页至少一个可视化元素
- 量化表达：每页至少一个数据支撑，数据需标注来源
- 禁止空话：禁用"全方位"、"深度赋能"、"生态化"等无实质内容词汇
- 每页附说话要点（Speaker Notes）
- 输出格式：提供完整PPT大纲（含每页标题 + 核心内容要点 + 图表建议 + Notes）

## I · 项目隐性背景【关键模块，越详细越好】

### I-1 客户内部决策结构
- 本次提案的最终拍板人：【填写】
- 直接对接人：【填写】
- 其他关键评审人：【填写】
- 各方优先级/权重：【填写】

### I-2 内部意见分歧
- 已知的内部分歧点：【填写】
- 提案需要帮助哪一方"赢"，还是需要平衡两方：【填写】
- 敏感话题（提案中需回避或谨慎处理的）：【填写】

### I-3 历史背景
- 此前同类提案/活动的结果：【填写】
- 客户对上一家供应商/方案的不满点：【填写】
- 本次提案与历史的关系：【填写】

### I-4 潜在顾虑与风险
- 客户已明确表达的顾虑：【填写】
- 你判断客户未说出口但可能存在的顾虑：【填写】
- 本次提案的竞争情况：【填写】

### I-5 软性信息
- 客户喜欢的表达风格：【填写】
- 客户反感的表达方式：【填写】
- 其他有助于赢得提案的信息：【填写】

---

## J · 红线约束（固定，不可绕过）

1. **数据真实性**：所有引用数据必须标注来源。若无真实数据支撑，必须标注「[数据待补充]」，禁止编造任何数字。
2. **品牌安全**：竞品分析描述须客观中立，不得出现可能引发法律风险的竞争性声明。
3. **平台合规**：投放建议须符合各平台最新广告法规，不得出现广告法违禁词。
4. **保密边界**：涉及客户内部数据、未公开信息，在提案中仅作为策略依据，不得直接引用或暴露信息来源。
5. **可执行性**：所有策略建议必须在给定预算和时间范围内可落地。
```

## 风格库

### 风格 A · 暖色行动感
适用场景：大促节点、效果营销、销售冲刺
配色：主色 #FF5722（橙红）+ 辅色 #FFC107（金黄）+ 底色 #FFFFFF

### 风格 B · 欧美 Bold
适用场景：品牌提案、高端客户、创意方向展示
配色：主色 #1A1A2E（深蓝黑）+ 辅色 #E94560（玫红）+ 底色 #0F3460

### 风格 C · 科技感
适用场景：数字营销、数据分析、技术解决方案
配色：主色 #0D1B2A（深蓝）+ 辅色 #00B4D8（青蓝）+ 底色 #03045E

### 风格 D · 品牌色定制
适用场景：所有有明确VI规范的品牌提案
使用方式：在 B 模块输入品牌主色 HEX，AI 自动生成配套配色方案

### 风格 E · 专业咨询感
适用场景：战略汇报、述职、管理层演示
配色：主色 #003366（深蓝）+ 辅色 #CC0000（红）+ 底色 #FFFFFF
